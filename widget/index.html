RAG Chatbot — Widget demo

This page is a local manual-test harness for widget.js. The widget is loaded with a placeholder public_key. To test against a real chatbot, edit the data-public-key attribute below to point at one of your chatbots and reload.

Embed snippet for production:

<script
  src="https://your-host/widget/widget.js"
  data-public-key="wgt_..."
  data-api-base="https://your-host"
  async
></script>

Open the browser dev console: any failures (bad public_key, CORS mismatch) are logged there.
